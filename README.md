# To-Do List (Web)

## Overview
This is a simple To-Do List web application developed using HTML, CSS, and JavaScript. The application allows users to add, manage, and remove daily tasks through an easy-to-use interface.

## Features
- Add new tasks
- Delete completed tasks
- Simple and responsive interface
- Easy task management

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run

1. Download or clone this repository.
2. Open `index.html` in any web browser.
3. Start managing your tasks.
